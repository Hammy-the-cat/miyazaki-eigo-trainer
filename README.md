# 宮崎県 中学英語 教採対策トレーナー

宮崎県公立学校教員採用選考試験（中学校英語）対策のためのWeb学習アプリです。

## 収録内容

- **文法・語法**: 200問（時制・助動詞・仮定法・受動態・不定詞・動名詞・分詞・関係詞・比較・接続詞・前置詞・冠詞・代名詞・形容詞副詞・語彙・イディオム・構文・語法）
- **中学英語 学習指導要領**: 200問（教科目標・5領域・音声・語彙・文構造文法・言語活動・言語の働き・指導計画・改訂のポイント等）

合計 **400問**、4択・解説付き。

## 主な機能

- 2段階カテゴリ選択（大分類→細分野）
- 出題数選択（10/20/50/全問）
- 即時フィードバックと解説
- 苦手問題の自動記録と復習モード
- 正答率・解答内訳のレビュー

## ローカル開発

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開きます。

## ビルド（本番用）

```bash
npm run build
npm run start
```

## デプロイ

このプロジェクトは [Vercel](https://vercel.com) で簡単にデプロイできます。

1. GitHubリポジトリを作成してこのコードをpush
2. Vercelダッシュボードで「New Project」→ GitHubリポジトリをインポート
3. 設定はデフォルトのまま「Deploy」

## 技術スタック

- Next.js 14 (App Router)
- React 18
- スタイリング: インラインCSS（Google Fonts: Noto Serif JP）
